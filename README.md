# دفتر الحسابات — تطبيق PWA (مع تجربة + تفعيل)

تطبيق حسابات شخصي: وارد/صادر، محافظ، وديون — بعملات منفصلة (شيكل/دولار/دينار)، يشتغل أوفلاين وينزّل على شاشة الآيفون، مع **فترة تجريبية يومين** ونظام **تفعيل بكود**.

---

## أولاً: النشر (نفس طريقة AlWateen-Lab)
ارفع هذه الملفات **فقط** على مستودع GitHub بجذره، وفعّل GitHub Pages:
- `index.html` , `app.js` , `styles.css`
- `manifest.webmanifest` , `service-worker.js`
- `icon-180.png` , `icon-192.png` , `icon-512.png`
- `README.md`

> **مهم جداً:** لا ترفع ملف `keygen.html` على GitHub! (شرحه تحت)

ثم: Settings → Pages → Branch: `main` و `/(root)` → Save. الرابط بيطلع خلال دقيقة–دقيقتين.
على الآيفون: افتح الرابط بـ Safari → مشاركة → «إضافة إلى الشاشة الرئيسية».

---

## ثانياً: كيف يشتغل البيع والتفعيل

1. **الزبون** ينزّل التطبيق من رابطك → بيشتغل **يومين تجربة** (بيطلعله شريط «باقي X يوم»).
2. بعد ما يقرّر يشترك، **يحوّللك** المبلغ (بال بي / جوال بي / بنك… حسب اتفاقكم).
3. تفتح **keygen.html** عندك (ملف خاص) → تكتب **اسم الزبون** → بيطلعلك **كود تفعيل**.
4. ترسل الكود للزبون → بيفتح ⚙️ → «تفعيل» → بيدخل **نفس الاسم + الكود** → بينفتح التطبيق مدى الحياة.

> الكود مربوط بالاسم: كود زبون ما بيشتغل لاسم تاني، فمشاركته صعبة.

---

## ملف keygen.html (سرّي — لك وحدك)
- هذا الملف بيولّد الأكواد. **احتفظ فيه على جهازك فقط.**
- **لا ترفعه على GitHub ولا ترسله لأي حدا** — اللي عنده هالملف بيقدر يولّد أكواد مجاناً.
- لتشغيله: افتحه بأي متصفّح (يفضّل على كمبيوتر)، اكتب اسم الزبون، انسخ الكود.

---

## ⚠️ صراحة عن مستوى الحماية
هذا التطبيق **بدون سيرفر**، فالحماية **برمجية محلية**:
- توقف المستخدم العادي تماماً (وهو معظم زبائنك).
- لكن شخص تقني متمكّن ممكن يتجاوزها (يعدّل كود الصفحة أو يمسح بيانات المتصفّح ليعيد التجربة).
- للحماية **الحقيقية** (سيرفر يتحقق من الاشتراك + تسجيل دخول + دفع آلي) بتحتاج باك-إند مثل Firebase — خطوة مناسبة لاحقاً.

**النصيحة:** ابدأ هيك (تفعيل يدوي)، جرّب تبيع لـ 10–20 شخص، ولو في طلب فعلي ننتقل لـ Firebase.

---

## ملاحظات عامة
- بيانات الزبون محفوظة على جهازه فقط — انصح زبائنك ياخدوا **نسخة احتياطية** من ⚙️.
- لو حدّثت التطبيق، غيّر رقم النسخة في service-worker.js (daftar-v1 → daftar-v2).
- المفتاح السري LIC_SECRET موجود في app.js و keygen.html — لو بدّلته، بدّله بنفس القيمة في الملفين معاً وإلا الأكواد ما بتتطابق.
